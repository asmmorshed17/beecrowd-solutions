# BeeCrowd Solutions (C Programming)

This repository contains my solutions to problems from [BeeCrowd](https://www.beecrowd.com.br/) (formerly URI Online Judge), implemented in **C programming language**.  

The problems are categorized by difficulty and topic to help learners and programmers explore algorithms, data structures, and problem-solving techniques.  

## 🔹 Repository Structure
- `/beginner` – Basic problems for beginners  
- `/ad-hoc` – Ad-hoc problem-solving challenges  
- `/math` – Mathematics and number theory problems  
- `/strings` – String manipulation challenges  
- `/others` – Miscellaneous problems

## 🔹 How to Use
- Explore solutions by topic or difficulty  
- Compile each `.c` file using a C compiler (e.g., `gcc filename.c -o output`)  
- Run the executable and input the sample test cases provided in the problem statement
